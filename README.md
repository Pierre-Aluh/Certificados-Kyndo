# Certikyndo

Landing page institucional desenvolvida com foco em performance, SEO e evolução para arquitetura de produto SaaS.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- ESLint

## Estrutura principal

- `src/app`: App Router, layout global e rota da landing
- `src/features/landing`: conteúdo e composição da página

## Execução local

```bash
npm install
npm run dev
```

Aplicação disponível em `http://localhost:3000`.

## Qualidade

```bash
npm run lint
npm run typecheck
npm run build
```
